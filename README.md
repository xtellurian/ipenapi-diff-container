# Openapi-diff-container

A docker container for the Atlassian OpenAPI Diff tool.

All credit to Atlassian. You can see the source tool [here on BitBucket](https://bitbucket.org/atlassian/openapi-diff/src/master/)


## Usage

```sh
docker run xtellurian/openapi-diff-container <OPEN API SPEC 1> <OPEN API SPEC 2>
```
